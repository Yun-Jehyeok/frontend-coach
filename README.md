# Frontend Coach

프론트엔드 코칭 서비스로, OpenAI와 병합되어 프론트엔드의 기초를 학습할 수 있는 플랫폼입니다.

## 기능(Features)

### **학습 시스템**

유저는 Web, HTML, CSS, JavaScript에 대한 기초 내용을 학습 할 수 있으며, 중간중간 퀴즈를 풀이하며 웹의 기초를 이해할 수 있습니다.

### **AI 답변**

퀴즈에 대한 답변은 AI가 평가를 진행합니다.

### **티어**

유저는 **Iron~Challenger**로 나뉘며 문제를 풀 때마다 티어가 상승합니다.

## Stack

Next.js, Next.js Server, Vercel DB
